Тег picture

Элемент, помогающий адаптировать изображение под любую ширину окна браузера. Также изменения изображений.

Также, элемент picture можно использовать для замены форматов изображений для конкретных бразуеров. Например, если изначальное изображение в элементе img в формате PNG, в значений source можно указать, например, формат изображения svg, если бразуер поддерживает такой формат - оно отобразится в svg. Формат пишем в атрибуте type
